# LR2LIFT

LR2のスキンCSVを編集して、LIFT（判定ラインの高さ調整）を簡単に設定できるツールです。

## 概要

LR2のスキンCSVには、各UI要素のY座標が定義されています。LIFTとはその判定ライン周辺を画面上方向に移動させることで、プレイ中の視認性を改善するカスタマイズです。

通常はCassavaなどのスプレッドシートエディタでCSVを手動編集する必要がありますが、このツールを使うとGUIで値を入力するだけで自動的に調整できます。

## 機能

- スキンCSVをGUIで開いてLIFT値・判定値を入力するだけで自動調整
- セクションのコメントを解析して対象行を自動検出（手動マーキング不要）
- 実行前にプレビューで変更内容を確認可能
- 再実行時は差分計算で累積しない（何度でも微調整可能）
- 初回実行時に自動でバックアップ（`.bak`）を作成

## 使い方

### 実行方法

**Python環境がある場合：**
```
python lr2lift.py
```

**Windows .exe を使う場合：**  
[Releases](../../releases) からダウンロードした `LR2LIFT.exe` をダブルクリック。

### 操作手順

1. **「開く...」** ボタンでスキンの CSV ファイルを選択
   - 例: `LR2files\Theme\スキン名\play\csv\AC_1p.csv`
2. **LIFT値**（レーン・判定ライン等を上げるpx数）を入力
3. **判定表示 追加値**（ジャッジ表示・FAST/SLOWをさらに上げるpx数）を入力
4. プレビューで変更行と調整後Y座標を確認
5. **「実行して保存」** をクリック

LR2を起動したまま値を変えながら何度でも再実行して調整できます。

### LIFT値の目安

| 設定 | 意味 |
|------|------|
| LIFT値 | レーン・判定ライン・ノート等を上げるpx数 |
| 判定表示 追加値 | ジャッジ表示・FAST/SLOW・ghost をさらに上げるpx数（LIFT値に加算） |

例: LIFT値=150, 判定表示=30 の場合
- レーン等: 150px上移動
- ジャッジ表示等: 180px上移動

## 自動検出されるセクション

| フラグ | 対象セクション |
|--------|---------------|
| L（LIFT） | レーン / 判定ライン / 判定ラインエフェクト / レーザー / 小節線 / ノート / ボム / LN EFFECT |
| LS（LIFT+判定） | ジャッジ表示 / FAST・SLOW / ghost typeA/B/C |

> スキンによってコメントの記述が異なる場合、検出されないセクションがあることがあります。その場合はCSVの列25に手動で `L` / `LS` / `S` を記入することで明示的に指定できます（Cassavaマクロと同じ形式）。

## バックアップについて

初回実行時（未編集のCSVに対して初めて実行したとき）に自動で `ファイル名.csv.bak` を作成します。すでに `.bak` が存在する場合は上書きしません。

元に戻したい場合は `.bak` ファイルをリネームしてください。

## Windows .exe のビルド方法

Windowsマシン上で以下を実行してください：

```
pip install pyinstaller
pyinstaller --onefile --windowed lr2lift.py --name LR2LIFT
```

`dist\LR2LIFT.exe` が生成されます。Python不要で単体動作します。

## 動作確認済みスキン

- WMII_FHD

## 動作環境

- Python 3.8 以上（.exeはPython不要）
- tkinter（通常Pythonに同梱）
- Windows（LR2自体がWindows専用）
